第三天git分支笔记

指令总结

将本地目录变成git目录：git init

将工作区文件添加到暂存区：git add -A

提交，从暂存区到仓库：git commit -m "注释"

查看当前状态：git status

查看内容：cat 文件名

删除文件：rm 文件名

删除文件夹：rmdir 文件夹名

查看目录下所有的东西：ls

清屏：clear

从命令行推送现有存储库

推送路径 建立与github的联系：git remote add origin 路径

推送到远程：推送到远程：git push -u origin master **注释**只有第一次需要git push -u origin master,以后就不需要了，直接git push 就ok了

分支指令

创建分支：git branch 名称

查看分支：git branch

切换分支：git checkout 名称

创建+切换分支：git checkout -b 名称

合并某分支到当前分支：git merge 名称

删除分支：git branch -d 名称

完毕
